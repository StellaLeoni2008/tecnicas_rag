## 📌 Sobre
Este projeto foi desenvolvido como parte das atividades do curso **Alura – LangChain: Técnicas Avançadas de RAG**. O objetivo do curso é aprofundar o entendimento sobre arquiteturas de Retrieval-Augmented Generation (RAG), explorando técnicas para melhorar a recuperação de informações e a qualidade das respostas geradas por modelos de linguagem.
Durante o projeto, foi construído um pipeline capaz de carregar documentos em PDF, processá-los, gerar embeddings e armazená-los em um banco vetorial, permitindo que modelos de linguagem consultem essas informações para responder perguntas de forma mais contextualizada.
O sistema utiliza LangChain para orquestração, Ollama para geração de embeddings e execução do LLM, e Milvus como banco de dados vetorial. Além disso, foram exploradas técnicas avançadas de recuperação de contexto, como HyDE (Hypothetical Document Embeddings), que gera uma resposta hipotética para melhorar a busca semântica nos documentos.

## 🧠 Conceitos Fundamentais Praticados
Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes de IA aplicada a sistemas de busca e geração de respostas, incluindo:
- Carregamento e processamento de documentos com DirectoryLoader
- Divisão de documentos em chunks utilizando CharacterTextSplitter
- Geração de embeddings semânticos para representar os textos
- Armazenamento e busca vetorial utilizando Milvus
- Uso de modelos de linguagem locais com Ollama
- Construção de pipelines com LangChain
- Implementação de técnicas avançadas de recuperação como HyDE
- Estruturação de prompts e chains para geração de respostas
- Integração entre LLM, embeddings e banco vetorial para sistemas RAG
